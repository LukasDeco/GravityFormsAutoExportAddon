With this plugin you can schedule automatic exports of your Gravity Forms entry data!
 
== Description ==
 
The AutoExport Gravity Forms Add-on comes with some cool features:
	- Select only the fields that you want to include in your export.
	- Create a fully custom export interval of whatever number of hours, days, weeks, and even months that you want. Want your export every 2 hours? You can do that. Want it every 3 weeks? You can do that too.
	- Set the export to be sent to multiple email addresses.

 
== Installation ==
 
This section describes how to install the plugin and get it working.
 
e.g.
 
1. Upload the 'AutoExport-CSV-GF' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Gravity Forms form you want to export and go to the settings menu.
4. Go to the 'AutoExport CSV' tab and configure your automatic export.
